# Nodejs_Brukermanual_Nettside

URL: [https://nodejs-brukermanual-nettside.herokuapp.com/](https://nodejs-brukermanual-nettside.herokuapp.com/)

## Production

### Install Dependencies
```bash
  npm install
```

### Start App
```bash
  npm run start
```

## Development

### Install Dependencies
```bash
  npm install
```

### Run Sass Compiler
```bash
  npm run sass
```

### Run Dev Server
```bash
  npm run dev
```
